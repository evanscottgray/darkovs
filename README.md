# darkovs
dark theme for ovs docs

# Install
1. Grab a copy of [Stylish](https://chrome.google.com/webstore/detail/stylish/fjnbnpbmkenffdnngjfgmeleoegfcffe).
2. Install the [DarkOVS Theme](https://userstyles.org/styles/114055/darkovs)!
